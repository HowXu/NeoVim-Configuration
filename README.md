# 🚀 neovim - LazyVim 个人配置

基于 [LazyVim](https://github.com/LazyVim/LazyVim) 的个性化 Neovim 配置。

## 特性

- **主题**: cyberdream + transparent
- **补全**: blink.cmp（Tab 接受补全，Enter 不自动补全）
- **LSP**: 优化某些 LSP 的表现
- **Git**: Neogit（`Space gg` 打开）
- **代码浏览**: Glance（`gp` 查看定义）、Neominimap（代码缩略图）
- **包裹**: mini-surround（`gsa` 添加，`gsd` 删除）
- **时间追踪**: WakaTime

## 定制

- 4 空格缩进，禁用自动换行 / 自动折叠
- `q` 键禁用（防误触录制宏）
- 自动保存（2 秒延迟）
- 块状光标
- Windows 下默认 shell 为 nushell
- Neovide GUI 支持
