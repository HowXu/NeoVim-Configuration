# 快捷键速查

## 通用

| 按键 | 功能 |
|------|------|
| `Space` | 前缀键 |
| `q` | 禁用（防误触录制宏） |
| `s` | 快捷搜索文件 |
| `Space ff` | 全局文件名搜索 |
| `Space sg` | 全局 grep 内容搜索 |

## 文件树 (neo-tree)

| 按键 | 功能 |
|------|------|
| `Space e` | 打开 / 关闭文件树 |
| `j` / `k` | 上下移动 |
| `h` / `l` | 折叠 / 展开目录 |
| `a` | 新建文件 / 目录 |
| `H` | 显示 / 隐藏文件（含未追踪） |
| `R` | 刷新文件树 |

## 编辑

| 按键 | 功能 |
|------|------|
| `gg` | 跳转到行首 |
| `G` | 跳转到行末 |
| `ggVG` | 全选 |
| `y` | 复制 |
| `p` | 粘贴 |
| `u` | 撤回 |
| `<C-r>` | 反撤销 |
| `V` | 选中当前行 |
| `gsa` | 选中后包裹 `{}` `[]` `""` `''` `()` |
| `gsd` | 删除包裹 |

## 移动

| 按键 | 功能 |
|------|------|
| `<C-f>` / `<C-b>` | 大段翻页（下 / 上） |
| `<S-l>` / `<S-h>` | 切换标签页缓冲区 |

## 窗口管理

| 按键 | 功能 |
|------|------|
| `w` + 方向键 | 分割与移动焦点 |
| `Space \|` | 右分屏 |
| `<C-h>` / `<C-l>` | 直接跳转左右窗口 |
| `<C-w>q` | 关闭当前窗口 |

## LSP

| 按键 | 功能 |
|------|------|
| `K` | 查看函数定义 (hover) |
| `KK` | 固定文档浮窗 |
| `gd` | 跳转到定义 |
| `gp` | 弹窗预览定义 (Glance) |
| `Space cf` | 格式化代码 |
| `Space ca` | 修复导入 (code action) |
| `Space xx` | 打开诊断列表 |

## 终端

| 按键 | 功能 |
|------|------|
| `:term` | 在缓冲区打开终端 |
| `Space ft` | 在下方分屏打开终端 |
| `i` | 进入终端模式 |
| `<C-\><C-n>` | 退出终端模式 |

## Git (Neogit)

| 按键 | 功能 |
|------|------|
| `Space gg` | 打开 Neogit UI |
| `s` | 暂存文件 (stage) |
| `P` | 推送 (push) |
| `p` | 拉取 (pull) |
| `cc` | 快速提交通知 |
| `<C-c><C-c>` | 确认提交 |

## Buffer

| 按键 | 功能 |
|------|------|
| `Space bd` | 关闭当前标签页 |
| `:e` | 打开目录 / 文件 |

## 补全 (blink.cmp)

| 按键 | 功能 |
|------|------|
| `<Tab>` | 接受补全项 |
| `<CR>` | 仅换行，不自动补全 |
| `<Up>` / `<Down>` | 选择上 / 下一项 |
